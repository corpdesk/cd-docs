ASDAP Data Manager - User Manual

The ASDAP Data Manager is a step-by-step wizard designed to help you create, manage, and control cooperative data efficiently. Follow the guide below to make the best use of this tool.

Overview

The Data Manager consists of three main features:

  • Creating New Data: Associate data batches with unique references.
  • Data Filtering: Control access to data using filters.
  • Data Submission: Enter statistical details about SACCOs.
Step 1: Select or Create a Reference

Every data batch must be linked to a reference. You can either select an existing reference or create a new one.

1.1 Select an Existing Reference

Use the dropdown to choose a reference. Once selected, details of the reference will be displayed, including:

  • ID: The unique identifier for the reference.
  • Name: The name of the reference.
1.2 Create a New Reference

If no existing reference fits, you can create a new one:

  1. Click the Create New Reference button.
  2. Fill in the required fields:
    • Name: Enter a name for the reference.
    • Description: Provide a description.
  3. Click Save Reference to proceed or Cancel to discard changes.
Step 2: Datasource Info

Define the type, geographic scope, and time period for your data.

2.1 Select Data Source Type

Use the Source Type dropdown to specify the type of data (e.g., `Continental SACCO`, `National Apex`).

2.2 Configure Geographic Information

Based on the selected source type, you may need to specify:

  • Continent: Choose a continent for `Continental SACCO`.
  • Country: Select a country for `National Apex` or `Microfinance SACCO`.
  • SubRegion: Optionally select a subregion if required.
2.3 Configure Temporal Information

Choose how to specify the time period for your data:

  • Year: Enter the year.
  • Date Selector: Use the calendar input to select a specific date.
Step 3: Statistical Data

Enter the cooperative statistics in this step. The required fields include:

  • Number of SACCOs: Total SACCOs in the dataset.
  • Number of Members: Total members in the SACCOs.
  • Saves & Shares: Savings and shares in USD.
  • Loans: Loans issued in USD.
  • Assets: Total assets in USD.
  • Penetration: Membership penetration percentage.
  • Reserves: Reserves in USD.
  • WOCCU Membership: Indicate if the SACCO is a WOCCU member.
Managing Filters

Filters control access to data batches. You can create, enable, disable, or delete filters as needed.

Create a Filter
  1. Click the Public Filter button on Step 1.
  2. Use the Filter Wizard to define access rules.
  3. Submit the filter.
Enable, Disable, or Delete Filters

Manage existing filters through the Filter Wizard:

  • Enable: Activate the filter to enforce rules.
  • Disable: Temporarily deactivate a filter.
  • Delete: Remove a filter permanently.
Best Practices
  • Ensure all data is tagged with a unique reference for easy tracking.
  • Select appropriate date granularity (Year or Date) for each dataset.
  • Use the wizard’s built-in validation to avoid errors.
  • Review and manage filters regularly for optimal data security.
